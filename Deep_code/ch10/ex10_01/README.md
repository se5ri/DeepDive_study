- 객체 리터럴의 닫는 중괄호 뒤에는 세미콜론을 붙인다

### 자바스크립트의 객체 생성 방식

- 클래스를 먼저 정의하고 new 연산자와 함께 생성자를 호출 할 필요 X → 리터럴로 객체 생성
- 객체 리터럴에 프로퍼티를 포함시켜 객체를 생성함과 동시에 프로퍼티를 만들수도 있고, 객체를 생성한 이후에 프로퍼티를 동적으로 추가 가능
- 객체 리터럴 외의 객체 생성 방식은 모두 함수를 사용해 객체를 생성
